# NoeAI Free Claude Code

Free interactive course, skills, templates, and gifts for Claude Code. Go from zero to a fully personalized AI system for your business in under an hour.

Built by [@noevarner.ai](https://instagram.com/noevarner.ai).

## One-Command Install

```bash
curl -fsSL https://raw.githubusercontent.com/noe-varner/NoeAI-Free-Claude-Code/main/install.sh | bash
```

This installs everything — skills, course, templates, and gifts. Re-run anytime to get updates.

## The Course (6 Lessons, ~1 Hour)

An interactive course that runs INSIDE Claude Code. Each lesson has a companion video + a hands-on skill that walks you through everything step by step.

| # | Lesson | What You'll Do |
|---|--------|---------------|
| 1 | **Personal AI Business Automator** | Claude learns YOUR business — name, audience, voice, tools |
| 2 | **Build Your First Skill** | Create a custom slash command from scratch |
| 3 | **Connect Your Apps (MCP)** | See Claude research your real tools live |
| 4 | **Your AI Team (Agents)** | Spawn multiple Claudes working in parallel |
| 5 | **Build Something Real** | Claude builds a real web page for your business |
| 6 | **The Full System** | Victory lap + see the complete picture |

**Start the course:** Open Claude Code (`claude`) and type `/noeai:lesson-1`

### Gifts Unlocked During the Course

Each lesson unlocks a free gift:

| Lesson | Gift |
|--------|------|
| 1 | **30 Prompts That Print Money** — copy-paste business prompts |

| 4 | **5 Agent Workflows** — multi-agent recipes |
| 5 | **Business-in-a-Box** — 10 mega-prompts that build complete systems |
| 6 | **Cheat Sheet** — full course reference on one page |

## How It Works

1. **Install** with one command (above)
2. **Open Claude Code** in your terminal: `claude`
3. **Start the course:** type `/noeai:lesson-1`
4. **Or use skills directly:** `/noeai:summarize`, `/noeai:repurpose`
5. **Re-run the installer** anytime to get updates
